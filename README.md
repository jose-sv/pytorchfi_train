# Simple training script

Checkout the main repository: https://github.com/pytorchfi/pytorchfi

```bash
pip install -r requirements.txt
python main.py
# CUDA_VISIBLE_DEVICES=2 python main.py  # to specify GPU id to ex. 2
```
